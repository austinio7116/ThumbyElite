ThumbyElite — Windows
=====================

An Elite/MechWarrior-style space sim. Trade, fight, salvage and outfit your
way from a junker to a dreadnought across an infinite procedural galaxy.

RUN
  Double-click ThumbyElite.exe. Keep SDL2.dll in the same folder.
  Your save (thumbyelite.sav) and settings are written next to the .exe.

CONTROLS
  Keyboard:   W/A/S/D = pitch/yaw,  . = fire,  , = secondary,
              Shift = LB (roll / cycle target),  Space = RB (throttle /
              flight-assist / boost),  Enter = menu,  Esc = quit.
  Gamepad:    auto-detected. Left stick fly, right stick X roll + Y throttle,
              A/B fire/secondary, RT fire, bumpers = LB/RB, Start = menu.
  HOTAS:      auto-detected. Stick = yaw/pitch, twist = roll, lever = throttle,
              trigger = fire. (See the in-game SETTINGS for aim sensitivity.)

That's it — no installer, no dependencies beyond SDL2.dll (included).
